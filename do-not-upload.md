## Summary

| Path | Upload? | Reason |
|---|---|---|
| `backend/*.py` | YES | Application source code |
| `static/**` | YES | Frontend HTML/CSS/JS |
| `app.py` | YES | Entry point |
| `requirements-ui.txt` | YES | Dependencies |
| `README.md` | YES | Documentation |
| `USER-GUIDE.md` | YES | User guide |
| `do-not-upload.md` | YES | This reference |
| `backend/__pycache__/` | NO | Auto-generated bytecode |
| `workspace/` | NO | All user data, trained models, and logs |

## Why `workspace/` Must Be Excluded

- **`data/`** — Contains copyrighted audio files belonging to the user
- **`datasets/`** — Generated from user data, can be rebuilt with the Convert button
- **`configs/`** — Written dynamically at training launch from UI settings
- **`exps/`** — Training checkpoints (multi-GB), logs, and generated audio; all user-specific

## Recommended .gitignore

```gitignore
# Python
__pycache__/
*.pyc
*.pyo

# Workspace (all user data and training outputs)
workspace/

# OS files
.DS_Store
Thumbs.db
desktop.ini

# IDE
.vscode/
.idea/
```
